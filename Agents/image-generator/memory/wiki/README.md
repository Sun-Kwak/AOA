# Agent Wiki

## Purpose

이 에이전트가 실행 중 발견한 실수, 해결책, 패턴을 기록한다.

공용 에이전트는 여러 프로젝트에서 재사용되므로, 학습이 누적되면 전체 품질이 향상된다.

---

## Wiki 파일

- **execution_patterns.md**: 실행 패턴, 입력 검증 실수
- **api_errors.md**: fal.ai API 에러, 타임아웃, 인증 실패
- **output_formats.md**: 이미지 형식 오류, 저장 경로 실수

---

## 사용 규칙

### 에이전트 시작 시
1. 모든 Wiki 문서 읽기
2. 기록된 실수 패턴 확인
3. 회피 전략 적용

### 에이전트 완료 후
1. 새로운 실수 발견 시 즉시 기록
2. 패턴 번호 부여
3. 회피 전략 명시

---

## 누적 효과

- 이 에이전트를 사용하는 모든 프로젝트가 혜택
- 동일 실수 반복 방지
- fal.ai API 에러 처리 개선
- 이미지 품질 향상
