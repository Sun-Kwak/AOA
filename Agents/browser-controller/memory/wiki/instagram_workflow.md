# Instagram Workflow

플랫폼별 배포 플로우는 실제 실행 시 누적됩니다.

---

## Format

```markdown
## 발생일: YYYY-MM-DD
## 상황: 이미지 업로드
## 플로우:
1. Navigate: https://instagram.com/create
2. Selector: .create-button
3. Wait: 2s
4. Upload: input[type="file"]
...

## 변경 이력:
- YYYY-MM-DD: 설명
```
