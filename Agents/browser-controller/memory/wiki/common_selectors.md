# Common Selectors

여러 플랫폼에서 공통으로 사용되는 셀렉터 패턴.

---

## Format

```markdown
## [Pattern-001] 업로드 버튼

**발생일**: YYYY-MM-DD  
**플랫폼**: Instagram, TikTok  
**셀렉터**: 
- Instagram: .create-button, .upload-icon
- TikTok: .upload-button, .create-post

**변경 이력**:
- YYYY-MM-DD: 설명
```
